# myGFXhat
Display easily on GFXHat for Raspberry

## Installation

### Auto Install
curl https://get.pimoroni.com/gfxhat | bash

### Manual Install
pip3 install gfxhat

Issue with async 
pip3 install --upgrade cap1xxx
pip3 install --upgrade sn3218

python -m venv venv 
source venv/bin/activate
python3 mygfxhat/main.py
